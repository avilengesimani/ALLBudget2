About the App A.L.L Budgeting is an Android mobile application built using Kotlin that allows users to track expenses, set budgets, and analyze spending by category.

The app is designed to help users manage personal finances efficiently with a simple and user-friendly interface

The Features

Login system
Add expense entries
Category management
Date, start & end time tracking
Optional receipt photo capture
Monthly spending summary
Category-based total calculations
Budget goal tracking (min & max)
Bottom navigation system
Tech Stack

Kotlin (Android)
Android SDK (API 24+)
Room Database (SQLite) / In-memory store (prototype)
XML UI Design
GitHub Actions CI/CD
How to Run

Clone repository
Open in Android Studio
Sync Gradle
Run on emulator or physical device
APK File

Download from GitHub Actions artifacts after successful build.
